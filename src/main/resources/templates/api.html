XML:

Use following path to use api:

to get all animals use /api/xml/animals

to get one animals use /api/xml/animals/"animal id"

to get all animal types use /api/xml/animaltypes

to get one animal type use /api/xml/animaltypes/"animaltype id"

to get all users use /api/xml/users

to get one user use /api/xml/users/"user id"

JSON:

Use following path to use api:

to get all animals use /api/json/animals

to get one animals use /api/json/animals/"animal id"

to get all animal types use /api/json/animaltypes

to get one animal type use /api/json/animaltypes/"animaltype id"

to get all users use /api/json/users

to get one user use /api/json/users/"user id"
